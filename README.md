# المخطط اليومي الذكي — نسخة الأندرويد

## طريقة الحصول على ملف APK

### الطريقة الأسهل: GitHub Actions (مجاناً)

1. **أنشئ حساب GitHub** (إذا لم يكن لديك): https://github.com
2. **أنشئ مستودع جديد** اسمه `smart-planner`
3. **ارفع محتوى هذا المجلد** إلى المستودع
4. **اذهب إلى Actions** في المستودع ← سيبدأ البناء تلقائياً
5. **انتظر 15-25 دقيقة** ← ثم حمّل الـ APK من قسم "Artifacts"
6. **ثبّته على موبايلك** عبر USB أو مباشرة

---

### البنية:
```
planner_apk/
├── main.py          ← التطبيق الرئيسي (Kivy)
├── db.py            ← طبقة قاعدة البيانات
├── buildozer.spec   ← إعدادات البناء
└── .github/
    └── workflows/
        └── build.yml  ← بناء تلقائي
```

### تفعيل التثبيت من مصادر غير معروفة:
الإعدادات ← الأمان ← "مصادر غير معروفة" ← تفعيل
